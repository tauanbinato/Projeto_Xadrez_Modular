Fala time, esse é nosso repositório para trabalharmos no projeto.

Passos Configurando Git + Visual 2015:

1 - Baixe o GitHub desktop
2 - Abra o software e clone para seu computador passando o link desse repositório ( Canto superior direito fica o link)
3 - Depois de clonado abra o Visual 2015 ou superior e procure a aba Team Explorer (Ao lado do Solution Explorer)
4 - Conecte-se na sua conta do github
5 - Após se conectar selecione clone dentro da aba GitHub
6 - Procure o repositorio que voce havia clonado pelo software
7 - Pronto o Visual e o GitHub estarão se comunicando
8 - Clique duas vezes no repositório no team explorer e abrira as opções de comando do git
9 - Clique em changes
10 - Toda vez que um arquivo for adicionado ou modificado na pasta do repositorio local (a pasta que clonou) o git vai mostrar qual arquivo e qual mudança foi feita ( Opções Changes)
11 - Sempre que você faz uma mudança você deve fazer um COMMIT antes de submeter a nuvem, onde ficara guardado com segurança. Um commit nada mais é do que um comentário que vai atrelado a mudança
12 - Quando fizer o commit você deve sincronizar o repositorio local com o repositorio do git(nuvem) esse processo chama-se PUSH
13 - Se você clicar duas vezes em cima da mudança você pode abrir uma janela na qual você pode ver as mudanças, o codigo que era vs o codigo novo
14 - Quando fizer o push as mudanças serão salvas no seu repositorio local e no repositorio do github.
